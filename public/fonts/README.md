# Custom fonts

Place your font files here. Supported formats: `.otf`, `.ttf`, `.woff`, `.woff2`

Fonts use **display names** (e.g. "Bold Title") rather than actual font names. Edit `src/css/fonts.scss` and `src/constants/fonts.ts` to add or rename.

## Bold Title (Helvetica Now Display)

1. Copy your Helvetica Now Display font files into this folder.

2. The app expects these filenames (edit `src/css/fonts.scss` if yours differ):
   - `HelveticaNowDisplay-Regular.otf` (400)
   - `HelveticaNowDisplay-Medium.otf` (500)
   - `HelveticaNowDisplay-Bold.otf` (700)

3. Appears as **Bold Title** in the font dropdown.

## Display Title (Bebas Neue)

1. `BebasNeue-Regular.ttf` is included.

2. Appears as **Display Title** in the font dropdown.
